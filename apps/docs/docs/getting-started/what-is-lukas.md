# What is $LUKAS?

$LUKAS is a revolutionary basket-stable cryptocurrency designed specifically for the Latin American market.

## Key Features

### Regional Stability
By combining multiple Latin American currencies, $LUKAS provides stability against individual currency fluctuations.

### Transparent Weighting
The currency basket is transparently weighted based on economic factors:
- GDP representation
- Trade volume
- Market liquidity

### Decentralized
Built on blockchain technology, $LUKAS operates without central authority control.

## Use Cases

- **Cross-border payments** within Latin America
- **Store of value** protected from single-currency volatility
- **DeFi applications** tailored for LatAm markets
